1. Start the Question Setting App

Open the software Bitvise SSH Client 6.31 and type the following code in the terminal to connect to the right folder:

cd /home/studentuser/code/server/uczlwj0

Then type the following code to start the httpServer:

node httpServer.js &

Next, you should connect to another folder by typing:

cd /home/studentuser/code/questions/uczlwj0

And type the following code to start the PhoneGap:

phonegap serve

Finally, type the following website in the browser explorer, you can see the interface of the Question Setting App and start to use it.

http://developer.cege.ucl.ac.uk:31275/

2. Find a location and design a quiz

Firstly, you need to go here: https://itouchmap.com/latlong.html to pick a location. This location is picked as an example. The name of this location is: Anatomy Building.

Once you have selected a location, click it on the map and you can see the coordinates of the location shown below the map.

Then, what you need to do is to design a quiz.

Here is an example:

Question: What university owns the Anatomy Building?

Answer 1: Kings College London

Answer 2: Imperial College London

Answer 3: University College London

Answer 4: the London School of Economics and Political Sciences

Correct Answer: c

Once you finish, all the information for creating a quiz has been prepared.

3. Creating a quiz in App

If you want to create a new quiz in this App, all the things you need to do is to fill the form in the sidebar with your designed quiz and click the “UploadData” button.

Once you click the button, you will see the prompt message. The following message means your quiz starts to be uploaded.

Next, you will see another prompt message which is used to check the data that you input.

If you see that the “…” below the button is changed to “Row Inserted”, that means you create the quiz successfully. If you do see any other information instead of “Row Inserted”, that means your operation is fail. Therefore, you need to check what might be wrong with your input and do the same job again.

4. Log out the App

If you have achieved all the operation for creating your quiz, you can close the window to close the App.

It would be better if you can close the PhoneGap by typing the following code in the terminal of Bitvise SSH Client 6.31:

“Ctrl” + C

Do not forget to close httpServer either:

fg 1

“Ctrl” + C
